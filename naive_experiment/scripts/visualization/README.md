# Visualization Tools

This directory contains scripts to visualize the results of the naive fine-tuning experiment.

## plot_metrics.py

Generates comparison plots for baseline vs fine-tuned model performance.

### Usage

```bash
python plot_metrics.py \
    --metrics-json ../results/metrics.json \
    --output-dir ./plots
```

### Arguments

- `--metrics-json`: Path to the metrics.json file generated by the evaluation script
- `--output-dir`: Directory to save the generated plots (default: `visualization_outputs`)

### Outputs

The script generates two plots:

1. **metrics_comparison.png**: Bar charts comparing average PSNR, SSIM, and LPIPS between baseline and fine-tuned models, with error bars showing standard error.

2. **metrics_distributions.png**: Histograms showing the distribution of each metric across all evaluated videos.

The script also prints a summary table to stdout with:
- Average metrics for baseline and fine-tuned models
- Standard errors
- Percentage improvements

### Requirements

```bash
pip install matplotlib numpy
```

Note: These dependencies should already be installed if you've completed the environment setup.
